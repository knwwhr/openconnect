# InsideJob - 통합 커리어 진단 플랫폼

## 프로젝트 개요
- **프로젝트명**: InsideJob - 통합 커리어 진단 플랫폼
- **위치**: `/home/knoww/career-app-integrated`
- **기술 스택**: React 18, Tailwind CSS, Supabase, JavaScript, HTML5
- **서비스 목적**: 커리어 진단 → 경쟁력 분석 → 자기소개서 작성을 하나의 플랫폼에서 제공
- **프로젝트 시작일**: 2025년 10월 10일

## 🎯 프로젝트 비전
**"취업을 준비하는 모든 사람들이 체계적으로 커리어를 진단하고 경쟁력을 높일 수 있는 원스톱 플랫폼"**

## 🔄 통합할 3개 서비스 분석

### 1. 커리어 진단기 (career-diagnosis)
**위치**: `/home/knoww/career-project/career-diagnosis`
**핵심 기능**: 
- 6개 직무그룹별 맞춤 진단 (기술/개발, 기획/전략, 마케팅/영업, 디자인/크리에이티브, 경영지원, 전문직)
- 280개 직무 지원 (KSCO + 워크넷 기반)
- 전공 기준 → 희망직무 기준 질문 시스템

**기술 구조**:
- Vanilla JavaScript 모듈화 구조
- 체계적인 Manager 패턴 (StateManager, UIManager, AnalysisManager 등)
- Supabase 연동 완료
- 완전 반응형 CSS

### 2. 취업경쟁력 계산기 (career-calculator) ✅ **구축 완료**
**위치**: `/home/knoww/career-project/career-calculator`
**핵심 기능**:
- **280개 직무 지원**: KSCO + 워크넷 기반 체계적 직무 분류
- **3단계 평가 시스템**: 기본정보(35점) + 직무역량(35점) + 시장수용도(30점) = 100점
- **6개 직무그룹별 특화 평가**: 기술개발, 기획전략, 마케팅영업, 디자인, 경영지원, 전문직
- **전문서비스/특수직군**: 금융권(은행/증권/보험/기타), 공기업, 영업/세일즈 등 신규 추가
- **한국적 현실 반영**: 컴활, 한글(HWP), MS Office 등 실무 중심 도구 평가
- **정원성장 메타포**: A급/B급 대신 🌳전문가→🌿숙련자→🌱성장→🌾도약→🌰새싹 단계
- **맞춤형 스킬 추천**: 부족한 영역별 구체적 개선 방향 제시
- **현실적 예측**: 등급별 예상 구직기간 및 취업성공률 제공

**기술 구조**:
- **React 18 CDN 기반**: 모던 JavaScript와 JSX 활용
- **단일 HTML 파일**: 7,000줄 규모의 완전한 SPA
- **Tailwind CSS**: 완전 반응형 모바일 퍼스트 디자인
- **모듈화된 로직**: 직무그룹별 독립적 평가 시스템
- **데이터 기반 추천**: 실제 채용시장 데이터 반영한 조언 시스템

### 3. 자기소개서 작성 툴킷 (self-intro-toolkit)
**위치**: `/home/knoww/career-consulting-brochure/self-intro-toolkit.html`
**핵심 기능**:
- 단계별 자기소개서 작성 가이드
- 실시간 작성 도움말
- 템플릿 기반 작성 지원

**기술 구조**:
- 단일 HTML 파일 구조
- 모던 CSS와 JavaScript

## 🏗️ 통합 전략

### 서비스 플로우 설계
```
사용자 진입 → 커리어 진단 → 경쟁력 계산 → 자기소개서 작성 → 결과 종합 리포트
```

### 통합 아키텍처 방향

#### 1. 기술 스택 통합
- **프론트엔드**: React 18 + Tailwind CSS (모든 서비스 통합)
- **백엔드**: Supabase (커리어 진단기 기존 연동 활용)
- **데이터 연동**: 진단 결과 → 계산기 → 자기소개서 데이터 흐름

#### 2. 데이터베이스 설계
```sql
-- 사용자 프로필
user_profiles (id, email, nickname, created_at)

-- 커리어 진단 결과
career_diagnosis_results (
    id, user_id, selected_job, job_group, 
    total_score, detailed_scores, recommendations
)

-- 경쟁력 계산 결과  
competitiveness_scores (
    id, user_id, diagnosis_id,
    total_score, skill_scores, improvement_areas
)

-- 자기소개서 데이터
self_intro_data (
    id, user_id, diagnosis_id,
    personal_statement, experiences, goals
)

-- 통합 리포트
career_reports (
    id, user_id, diagnosis_id, competitiveness_id, 
    self_intro_id, generated_at, report_data
)
```

#### 3. 컴포넌트 구조
```
src/
├── components/
│   ├── common/          # 공통 컴포넌트
│   ├── diagnosis/       # 커리어 진단
│   ├── calculator/      # 경쟁력 계산기
│   ├── toolkit/         # 자기소개서 툴킷
│   └── report/          # 통합 리포트
├── managers/            # 기존 Manager 패턴 활용
├── utils/               # 공통 유틸리티
└── data/                # 직무 데이터
```
